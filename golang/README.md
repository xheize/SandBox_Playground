# Go Projects

이 폴더는 Go (Golang) 프로젝트를 위한 공간입니다.

## 새 프로젝트 시작하기

```bash
# 새 프로젝트 폴더 생성
mkdir my-project
cd my-project

# Go 모듈 초기화
go mod init github.com/xheize/my-project

# 의존성 설치 예시
# go get github.com/gin-gonic/gin
```

## 프로젝트 예시

- `web-server/` - HTTP 웹 서버
- `cli-tool/` - 커맨드라인 도구
- `microservice/` - 마이크로서비스
- `go-experiment/` - Go 기능 실험

## 권장 도구

- go fmt: 코드 포맷팅
- go vet: 코드 검사
- golint: 린팅
- go test: 테스팅
