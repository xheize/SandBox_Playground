# JavaScript/Node.js Projects

이 폴더는 JavaScript 및 Node.js 프로젝트를 위한 공간입니다.

## 새 프로젝트 시작하기

```bash
# 새 프로젝트 폴더 생성
mkdir my-project
cd my-project

# Node.js 프로젝트 초기화
npm init -y

# 또는 TypeScript 프로젝트
# npx create-react-app my-app --template typescript
# npx create-next-app@latest my-app
```

## 프로젝트 예시

- `express-api/` - Express.js API 서버
- `react-app/` - React 애플리케이션
- `node-script/` - Node.js 스크립트
- `typescript-project/` - TypeScript 프로젝트

## 권장 도구

- npm/yarn/pnpm: 패키지 관리
- ESLint: 린팅
- Prettier: 코드 포맷팅
- Jest/Vitest: 테스팅
